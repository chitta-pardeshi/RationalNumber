# RationalNumber, RationalMatrix and RationalVector
Rational Number Represented As Ratio of BigInteger


            RationalMatrix A = new double[,] {
                { 4, 7 },
                { 2, 6 }
            };
            RationalMatrix B = A.InverseMatrix();
            System.Console.WriteLine(B.ToString());

                [
                 [ 3/5 , -7/10 ],
